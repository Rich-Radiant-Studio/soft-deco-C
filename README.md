# C端用户应用

基于 Vue 3 的现代化C端用户应用，提供完整的用户界面、购物车、订单管理等功能。

## 🏗️ 项目概述

这是一个独立的C端用户应用，包含登录注册功能和完整的用户端功能。

## 📁 项目结构

```
user-web-project/
├── packages/
│   └── user-web/           # C端用户应用 (端口: 8085)
│       ├── src/
│       │   ├── components/ # 组件
│       │   ├── views/      # 页面
│       │   ├── router/     # 路由
│       │   ├── stores/     # 状态管理
│       │   ├── utils/      # 工具函数
│       │   └── plugins/    # 插件
│       └── package.json
└── package.json           # 工作空间配置
```

## 🎯 功能特点

### 👤 用户功能
- **登录注册**: 完整的用户认证系统
- **游客模式**: 支持游客浏览部分内容
- **个人中心**: 用户信息管理
- **购物车**: 商品选择、数量调整、价格计算
- **订单管理**: 订单查看和管理
- **多语言**: 支持中英文切换

### 🛒 购物车功能
- 商品列表展示（图片、名称、规格、价格）
- 商品数量调整（加减按钮）
- 商品选择（复选框）
- 配送方式选择（标准配送、快速配送）
- 支付方式选择（VISA、PayPal、Apple Pay）
- 订单详情计算
- 收货地址管理

## 🛠️ 技术栈

### 核心技术
- **框架**: Vue 3 (Composition API, `<script setup>`)
- **构建工具**: Vite
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **国际化**: vue-i18n
- **HTTP客户端**: Axios
- **类型检查**: TypeScript

### UI 组件库
- **Bootstrap 5**: 基础样式框架
- **BootstrapVue 3**: Vue 3 组件库
- **Bootstrap Icons**: 图标库

## 📋 开发规范

### 命名规范
- **组件文件夹/文件**: 大驼峰 (PascalCase)
- **变量与函数**: 小驼峰 (camelCase)
- **常量**: 全大写下划线 (SNAKE_CASE)

### 响应式设计
- 使用 `clamp()` 函数实现响应式尺寸
- 支持桌面端和移动端
- 所有尺寸自动根据分辨率调整

### 代码组织
- 复杂逻辑抽离到 `useXXX` 组合式函数 (Composables)
- CSS 必须使用 scoped
- 必须定义 defineProps 和 defineEmits
